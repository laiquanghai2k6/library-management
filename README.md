## 🖥️ Tech Stack

- **Frontend:** JavaFX (Java)

## 🛠️ Tính năng chính

1. **Quản lý người dùng người dùng**
   - Thêm người dùng
   - Sửa người dùng
2. **Quản lý tài liệu**
   - Thêm tài liệu
   - Sửa tài liệu
   - Xóa tài liệu 
3. **Mượn/Trả tài liệu**
   - Tìm kiếm tài liệu
   - Tìm kiếm người dùng
   - Mượn tài liệu
   - Trả tài liệu
   - Đánh giá tài liệu
4. **Xử lí các trường hợp lỗi**
5. **Giao diện người dùng**
6. **Sử dụng đa luồng cải thiện UX**
7. **Có thể test các Service bằng JUnit**
